# WearWise — Impeccable Design Context

## Design Context

### Users
Single user: Gaurav Kumar (The Algothrim). Personal AI wardrobe app installed as a PWA on his Samsung phone. Used daily — quick morning outfit decisions, travel packing, mode-based suggestions. Context is always mobile, always him, no onboarding or discoverability needed.

### Brand Personality
**3 words: Personal. Instant. Effortless.**
This is a private tool, not a product. It should feel like an extension of the phone's native OS, not a standalone app.

### Aesthetic Direction
- **Theme:** AMOLED dark — pure `#000000` background, not dark gray. Matches the Samsung One UI 8.5 system aesthetic on his phone.
- **Accent:** Crimson `#E2335D` — matches his phone's personalization color. Non-negotiable.
- **Typography:** SamsungOne / SF Pro Display stack. Hierarchical, clean, no decorative type.
- **Feel:** Samsung One UI 8.5 native — generous padding, pill shapes, smooth transitions, matte flat cards, no glassmorphism, no glows, no gradients. Content-first.
- **Anti-references:** No fashion magazine aesthetic. No luxury editorial. No glassmorphic blur. No AI-generated gradient vibes.

### Design Principles
1. **One tap, done** — every core action (generate, add, browse) must be reachable in 1 tap from wherever you are.
2. **Content is the UI** — outfit cards and item photos are the visual heroes. The chrome stays quiet.
3. **Native feel** — transitions, spring physics, tap feedback, safe-area handling should all feel indistinguishable from a Samsung system app.
4. **No waste** — no empty states that apologize, no redundant labels, no decorative elements that don't earn their place.
5. **Personal over polished** — minor imperfections are fine. The app should feel tuned for one person, not designed for a public launch.
